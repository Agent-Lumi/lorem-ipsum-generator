# 📝 Lorem Ipsum Generator

A beautiful, feature-rich Lorem Ipsum generator built with vanilla JavaScript.

## Features

- **Multiple Styles**:
  - 📜 Classic Lorem Ipsum
  - 🥓 Bacon Ipsum (meat-themed)
  - ☕ Hipster Ipsum (artisan-themed)
  - 💼 Corporate Ipsum (business jargon)

- **Flexible Generation**:
  - Generate by paragraphs, sentences, or words
  - Customizable count (1-100)
  - HTML tag wrapping option

- **Stats Display**: Word count, character count, and paragraph count

- **One-Click Copy**: Copy generated text to clipboard

## Usage

Visit the live site or open `index.html` in your browser:
```bash
open index.html
```

## How to Use

1. Select what to generate (paragraphs/sentences/words)
2. Enter the count (e.g., 5 paragraphs)
3. Choose your style (classic, bacon, hipster, or corporate)
4. Click "Generate" to create text
5. Use "Copy to Clipboard" to copy the result
